## Χαρακτηριστικά

- **Επιλογή φακέλου** με ένα κλικ μέσω Directory Chooser
- **Prefix & Suffix** — πρόσθεσε κείμενο πριν ή μετά το όνομα
- **Find & Replace** — αντικατάστησε οποιοδήποτε τμήμα του ονόματος
- **Preview mode** — δες τις αλλαγές *πριν* εφαρμοστούν
- **Progress bar** για παρακολούθηση σε πραγματικό χρόνο
- **Recursive mode** — επεξεργασία υποφακέλων
- **Modern Dark Theme** — επαγγελματικό UI με JavaFX
- **Zero dependencies** — δεν χρειάζεται Java εγκατεστημένη

## Οδηγός χρήσης

### Βήμα 1 — Επέλεξε φάκελο
Κλικ στο **"Επιλογή Φακέλου"** και διάλεξε τον φάκελο με τα αρχεία σου.

### Βήμα 2 — Όρισε κανόνες
Συμπλήρωσε ένα ή περισσότερα από τα παρακάτω:
- **Prefix**: Κείμενο που θα προστεθεί *πριν* το όνομα (π.χ. `2024_`)
- **Suffix**: Κείμενο που θα προστεθεί *μετά* το όνομα (π.χ. `_final`)
- **Εύρεση / Αντικατάσταση**: Βρες και αντικατάστησε τμήμα του ονόματος

### Βήμα 3 — Preview
Κλικ στο **"👁 Preview"** για να δεις τα νέα ονόματα *χωρίς* να αλλάξει τίποτα ακόμα.

### Βήμα 4 — Μετονομασία
Αν είσαι ικανοποιημένος με το preview, κλικ στο **"✓ Μετονομασία"**.
