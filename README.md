# Bulk File Renamer

Το **Bulk File Renamer** είναι ένα ισχυρό, γρήγορο και ασφαλές εργαλείο για Windows, σχεδιασμένο να κάνει τη μαζική μετονομασία αρχείων μια απλή και ξεκούραστη διαδικασία.

---

## Κύρια Χαρακτηριστικά

* **Ταχύτητα:** Μετονομάστε εκατοντάδες αρχεία μέσα σε λίγα δευτερόλεπτα.
* **Live Preview:** Δείτε ακριβώς πώς θα φαίνονται τα αρχεία σας *πριν* οριστικοποιήσετε τις αλλαγές.
* **Ευέλικτοι Κανόνες:** Προσθήκη **Prefix & Suffix** με ένα κλικ.
* **Find & Replace** λειτουργία για γρήγορες διορθώσεις ονομάτων.
* **Zero Dependencies:** Δεν απαιτείται εγκατάσταση Java ή άλλων βιβλιοθηκών. Περιλαμβάνει όλα τα απαραίτητα αρχεία για άμεση εκτέλεση.
* **Modern UI:** Καθαρό, σκοτεινό (dark theme) περιβάλλον εργασίας, φιλικό προς τον χρήστη.
* **Recursive Mode:** Προαιρετική επεξεργασία αρχείων και σε υποφακέλους.
